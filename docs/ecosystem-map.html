⚙ Dev notes

Keeping this map true to the code.

Resync after the code changes

node tools/ecosystem-map-gen.mjs

Reconciles the curated model against the live Rust source: attaches real line numbers, fills missing text from doc-comments, and flags drift — STALE (a node points at renamed/removed code → shows a “?” badge), UNMAPPED (exported fns not yet on the map), ORPHAN (source files not on the map).

Just check, no write (CI)

node tools/ecosystem-map-gen.mjs --check

Edit the narrative — systems, connectors, payloads, seams — in docs/ecosystem-map.model.json (the source of truth). The embedded graph is generated between the <<ECOSYSTEM-GRAPH-START / END>> markers — don't hand-edit it.

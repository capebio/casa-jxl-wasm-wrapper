## Current Limitations (This Slice)
- ROI view paints may size the lightbox canvas to the sub-rect (view treats the ROI as current logical source).
- Editing (straighten/hist) will cause a second full decode (low pri) — protected but not zero-cost.
- Math is first-cut (center + simple bleed/ds); will need tuning from real runs.
- Progressive + region in libjxl is still full-then-crop (known limitation; `progressiveRegionDecode: false`).
- Mixed full/ROI for same URL uses URL-only dedup (first options win).

Effort only affects encode speed and file size — not image quality. Quality slider controls that. All levels at the same quality setting look identical; effort decides how efficiently the data is packed.

                        

                            
1⚡ LightningUltra-fast encode. Files ~25–35% larger than Squirrel. Use for draft batches where speed beats size.

                            
3🦅 FalconFast — roughly 3–5× faster than Hare. Good default for everyday shooting.

                            
5🐇 HareBalanced. Files ~5–10% smaller than Falcon.

                            
6🦘 WombatGood compression — slower than Hare, smaller files. Max practical effort for 20 MP+ images; higher efforts fall back here if they run out of WASM heap.

                            
7🐿 SquirrelBest compression — ~3–5× slower than Hare. Falls back to Wombat on images larger than ~12 MP due to WASM memory limits.
